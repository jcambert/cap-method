# LIVRABLE-084
# Grille d'amélioration CAP v1.2

**Version :** 1.0

---

## Objectif

Cette grille transforme les retours du pilote en décisions concrètes pour préparer une version améliorée de CAP.

Elle permet de classer les retours, prioriser les corrections et décider ce qui entre dans la version 1.2.

---

## Sources de retour

Les améliorations peuvent venir de :

- fiche retour participante ;
- fiche retour accompagnant ;
- journal pilote ;
- difficultés observées ;
- temps réel passé ;
- questions mal comprises ;
- supports non utilisés ;
- restitutions difficiles à produire.

## Niveaux d'impact

| Niveau | Définition |
|---|---|
| Faible | Amélioration de confort |
| Modéré | Améliore la clarté ou l'usage |
| Fort | Corrige une difficulté importante |
| Critique | Empêche ou bloque l'utilisation |
