# LIVRABLE-080
# Modèle de dossier participante CAP

**Version :** 1.0

---

## Objectif

Ce document propose une structure de dossier standard pour suivre une participante pendant tout le parcours CAP.

Il permet de centraliser les réponses, analyses, restitutions, hypothèses, décisions et documents de clôture.

## 00_admin

Ce dossier contient les informations de cadrage.

### cadre-accompagnement.md

À renseigner :

- prénom participante ;
- date de démarrage ;
- canal principal ;
- rythme prévu ;
- objectifs initiaux ;
- limites de l'accompagnement ;
- points de vigilance connus.

### objectifs-initiaux.md

Questions utiles :

- Pourquoi la participante démarre CAP ?
- Qu'espère-t-elle clarifier ?
- Quelle situation souhaite-t-elle améliorer ?
- Qu'est-ce qui serait un résultat utile pour elle ?

### suivi-rendez-vous.md

Tableau conseillé :

| Date | Étape | Sujet | Décision | Prochaine action |
|---|---|---|---|---|
| | | | | |

## 01_reponses_forms

Ce dossier archive les réponses par module.

Chaque fichier doit contenir :

- date de réponse ;
- lien éventuel vers Google Forms ;
- réponses principales ;
- remarques de contexte ;
- éléments à clarifier.

---

Ce dossier contient le travail privé de structuration de l'accompagnant.

|---|---|---|---|---|

|---|---|---|---|

### points-vigilance.md

Format conseillé :

|---|---|---|---|
| | | | |

### decisions-accompagnant.md

À utiliser pour noter :

- module à approfondir ;
- cahier envoyé ;
- restitution ajustée ;
- hypothèse abandonnée ;
- rythme modifié.

---

## 03_restitutions

Ce dossier contient les restitutions intermédiaires envoyées à la participante.

Règles :

- une restitution par module ;
- conserver la version envoyée ;
- noter les retours de la participante ;
- ajuster si nécessaire.

---

## 04_synthese_finale

Ce dossier contient :

- le brouillon de synthèse ;
- la version validée ;
- la grille de validation ;
- le retour de lecture de la participante.

La synthèse finale doit rester révisable.

---

## 05_suivi_post_cap

Ce dossier suit les actions après la clôture.

Il peut contenir :

- suivi à 2 semaines ;
- suivi à 1 mois ;
- révision à 3 mois ;
- bilan à 6 mois.

---

## Nommage recommandé

Utiliser des noms simples :

```text
2026-07-05-restitution-module-01.md
2026-07-12-restitution-module-02.md
2026-09-30-synthese-finale-validee.md
```

Éviter :

- les noms trop longs ;
- les caractères spéciaux ;
- les versions non datées ;
- les doublons.

---

## Confidentialité

Le dossier participante peut contenir des informations personnelles.

Recommandations :

- limiter l'accès ;
- anonymiser si partage externe ;
- ne pas publier sans accord ;
- éviter les données inutiles ;
- supprimer les informations sensibles non nécessaires.

## Résultat attendu

À la fin d'un accompagnement CAP, ce dossier doit permettre de retrouver facilement :

- ce que la participante a exprimé ;
- ce qui a été restitué ;
- les hypothèses retenues ;
- les scénarios explorés ;
- le plan d'action ;
- les prochaines étapes.
