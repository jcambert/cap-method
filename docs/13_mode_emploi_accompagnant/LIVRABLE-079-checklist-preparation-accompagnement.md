# LIVRABLE-079
# Checklist de préparation d'un accompagnement CAP

**Version :** 1.0

---

## Objectif

Cette checklist permet de préparer un accompagnement CAP avant de démarrer avec une participante.

Elle évite d'improviser le cadre, les outils et les supports.

## 2. Message de démarrage

Avant d'envoyer le premier formulaire, vérifier que la participante comprend :

- CAP n'est pas un test ;
- il n'y a pas de bonne ou mauvaise réponse ;
- elle peut répondre à son rythme ;
- elle peut demander une clarification ;
- elle reste libre de valider ou non les restitutions ;
- le parcours vise du concret.

Message prêt :

- [ ] Oui
- [ ] Non

## 9. Points de vigilance personnels

Avant de commencer, noter les points connus :

- disponibilité limitée ;
- fatigue ;
- situation émotionnelle sensible ;
- contrainte familiale ;
- contrainte professionnelle ;
- difficulté à écrire ;
- besoin de relances courtes ;
- besoin de temps.

## 10. Critères de démarrage

L'accompagnement peut démarrer si :

- [ ] la participante a compris le cadre ;
- [ ] le canal d'échange est défini ;
- [ ] le premier formulaire est prêt ;
- [ ] le dossier participante est créé ;
- [ ] la méthode de restitution est claire ;
- [ ] l'accompagnant dispose du temps nécessaire pour lire et restituer.
