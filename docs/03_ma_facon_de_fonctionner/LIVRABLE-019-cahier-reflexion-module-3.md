# LIVRABLE-019
# Cahier de réflexion – Module 3 : Ma façon de fonctionner

**Version :** 1.0

---

## Présentation

Ce cahier complète le formulaire **Ma façon de fonctionner**.

Le formulaire permet d'identifier tes préférences générales.

Ce cahier t'invite à les illustrer avec des situations concrètes.

L'objectif n'est pas de te définir une fois pour toutes.

L'objectif est de mieux comprendre les conditions dans lesquelles tu fonctionnes le mieux.

## Pour l'accompagnant

Cette section n'est pas destinée à la participante.

À l'issue de ce cahier, chercher à identifier :

- les conditions de réussite ;
- les contextes coûteux ;
- les préférences d'organisation ;
- les besoins de cadre ou d'autonomie ;
- les éléments à relier aux valeurs du Module 2.

Ne pas transformer ces préférences en traits fixes.

La restitution doit rester contextuelle et prudente.
