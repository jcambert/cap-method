1. Envoyer une réponse fictive dans un Google Forms.
2. Vérifier l'arrivée dans Google Sheets.
3. Reporter un élément dans une feuille d'analyse.
4. Créer une hypothèse.
5. Ajouter un point dans le tableau de bord.
6. Identifier un élément à reprendre en restitution.
