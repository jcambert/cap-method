# LIVRABLE-090
# Guide de création Google Sheets CAP

**Version :** 1.0

---

## Objectif

Ce guide explique comment créer et utiliser les Google Sheets CAP.

Les Google Sheets servent à centraliser les réponses, structurer l'analyse accompagnant, suivre les hypothèses, préparer les restitutions et piloter l'avancement du parcours.

---

## Principe général

CAP peut être utilisé avec deux organisations possibles :

1. **Un classeur par module** : plus simple au début, mais plus dispersé.
2. **Un classeur global CAP** : plus structuré, recommandé pour un accompagnement complet.

La recommandation CAP est d'utiliser un **classeur global** avec des onglets standardisés.

---

## Règle importante

Ne jamais modifier directement les feuilles de réponses brutes venant de Google Forms.

Créer toujours des feuilles séparées pour :

- analyse ;
- hypothèses ;
- croisements ;
- restitutions ;
- suivi ;
- décisions.

---

## Classeur global recommandé

Nom du fichier :

```text
CAP – Classeur global – Prénom participante
```

Pour un modèle réutilisable :

```text
CAP – TEMPLATE – Classeur global
```

---

## Structure minimale

Le classeur global doit contenir :

- Tableau de bord
- Index modules
- Réponses brutes
- Hypothèses
- Fils rouges
- Ressources
- Contraintes
- Scénarios
- Plan d'action
- Restitutions
- Journal accompagnant

---

## Méthode de création

### Étape 1 – Créer le classeur

Créer un nouveau Google Sheets.

Nommer le fichier selon la convention.

### Étape 2 – Créer les onglets de pilotage

Créer d'abord :

- Tableau de bord
- Index modules
- Journal accompagnant
- Hypothèses globales

Ces feuilles permettent de suivre tout le parcours.

### Étape 3 – Connecter les formulaires

Pour chaque Google Forms :

- connecter à une feuille de réponses ;
- nommer clairement l'onglet ;
- ne pas renommer les colonnes automatiquement créées sans raison ;
- reporter les noms techniques attendus si nécessaire.

### Étape 4 – Créer les feuilles d'analyse

Pour chaque module, créer une feuille d'analyse ou utiliser une feuille globale.

Exemple :

```text
M01 Analyse
M02 Analyse
M03 Analyse
```

### Étape 5 – Créer les feuilles transversales

Créer :

- Hypothèses globales
- Fils rouges
- Points de vigilance
- Scénarios
- Plan d'action
- Synthèse finale

---

## Feuilles recommandées

| Feuille | Usage |
|---|---|
| Tableau de bord | Vue globale de l'avancement |
| Index modules | Suivi des formulaires et restitutions |
| Réponses M00 à M11 | Réponses brutes |
| Analyse M01 à M11 | Lecture accompagnant |
| Hypothèses globales | Hypothèses transversales |
| Fils rouges | Éléments récurrents |
| Points de vigilance | Contraintes ou tensions |
| Scénarios | Comparaison des pistes |
| Plan d'action | Actions et échéances |
| Synthèse finale | Préparation du document final |
| Journal accompagnant | Notes de suivi |

---

## Bonnes pratiques

- Utiliser une couleur par famille d'onglets.
- Protéger les feuilles de réponses brutes.
- Figer la première ligne.
- Utiliser des filtres.
- Ne pas surcharger les formules.
- Garder les feuilles lisibles.
- Documenter les hypothèses.

---

## Couleurs recommandées

| Type de feuille | Couleur |
|---|---|
| Pilotage | Bleu |
| Réponses brutes | Gris |
| Analyse | Vert |
| Hypothèses | Violet |
| Risques / contraintes | Rouge ou orange |
| Restitutions | Jaune |
| Plan d'action | Bleu clair |

---

## Exploitation des réponses

Pour chaque module, extraire :

- éléments forts ;
- exemples concrets ;
- scores importants ;
- contradictions ;
- questions à approfondir ;
- hypothèses possibles ;
- éléments à reprendre en restitution.

---

## Suivi des hypothèses

Chaque hypothèse doit contenir :

- un identifiant ;
- une formulation prudente ;
- les modules sources ;
- les indices ;
- le niveau de confiance ;
- les éléments à confirmer ;
- la décision : garder, ajuster, abandonner.

---

## Contrôle avant usage réel

Avant d'utiliser le classeur avec une participante :

- [ ] Les onglets sont créés.
- [ ] Les formulaires sont connectés.
- [ ] Une réponse test a été envoyée.
- [ ] Le tableau de bord fonctionne.
- [ ] Les feuilles brutes sont protégées.
- [ ] Le dictionnaire de colonnes est disponible.
- [ ] Le journal accompagnant est prêt.

---

## Résultat attendu

À la fin de la création, l'accompagnant dispose d'un classeur capable de suivre un parcours CAP complet du premier formulaire à la synthèse finale.
