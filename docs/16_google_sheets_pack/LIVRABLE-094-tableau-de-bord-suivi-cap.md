# LIVRABLE-094
# Tableau de bord de suivi CAP

**Version :** 1.0

---

## Objectif

Ce document définit le tableau de bord de suivi CAP à intégrer dans le classeur global.

Il donne à l'accompagnant une vue claire de l'avancement du parcours, des réponses reçues, des restitutions à produire et des prochaines actions.

---

## Vue principale

Le tableau de bord doit permettre de répondre rapidement à ces questions :

- Quel module est en cours ?
- Quel formulaire a été envoyé ?
- Les réponses ont-elles été reçues ?
- L'analyse est-elle faite ?
- La restitution est-elle envoyée ?
- Quelle est la prochaine action ?
- Y a-t-il un point de vigilance ?

---

## Colonnes recommandées

| Colonne | Description |
|---|---|
| Module | Numéro et nom du module |
| Formulaire | Statut du formulaire |
| Réponses | Statut des réponses |

| Restitution | Statut de la restitution |

| Commentaire | Note accompagnant |

## Règles d'utilisation

Mettre à jour le tableau de bord :

- après l'envoi d'un formulaire ;
- après réception des réponses ;
- après analyse ;
- après envoi d'une restitution ;
- après retour participante ;
- avant de passer au module suivant.

Le tableau de bord doit permettre à l'accompagnant de piloter un accompagnement CAP sans se perdre dans les fichiers, les modules et les restitutions.
