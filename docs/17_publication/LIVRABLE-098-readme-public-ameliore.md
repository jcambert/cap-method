# CAP Method

## Parcours d'accompagnement personnel et professionnel

CAP est une méthode structurée pour accompagner une personne dans une démarche de clarification personnelle et professionnelle.

Elle aide à mieux comprendre :

- son parcours ;
- ses valeurs ;
- ses besoins ;
- ses motivations ;
- ses compétences ;
- ses talents ;
- son environnement favorable ;
- ses ressources et contraintes ;
- ses scénarios d'évolution ;
- son plan d'action.

---

## Objectif du dépôt

Ce dépôt contient les sources Markdown de la méthode CAP.

Il sert à :

- versionner les contenus ;
- préparer les supports d'accompagnement ;
- générer des exports DOCX/PDF ;
- créer des Google Forms ;
- créer des Google Sheets ;
- piloter des parcours terrain ;
- améliorer la méthode à partir des retours.

---

## Ce que CAP n'est pas

CAP n'est pas un test psychologique, un diagnostic, une thérapie ou un bilan réglementaire.

CAP ne décide pas à la place de la personne.

La méthode propose un cadre d'exploration, de restitution prudente et de mise en action progressive.

## Utilisation recommandée

1. Lire l'avertissement méthodologique.
2. Lire le mode d'emploi accompagnant.
3. Préparer le dossier participante.
4. Créer ou dupliquer les Google Forms.
5. Créer le classeur Google Sheets.
6. Envoyer les modules progressivement.
7. Produire les restitutions intermédiaires.
8. Construire la synthèse finale.
9. Tester le plan d'action.
10. Collecter les retours terrain.
