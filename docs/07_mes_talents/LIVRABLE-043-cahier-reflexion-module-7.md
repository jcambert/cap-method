**Version :** 1.0

## Présentation

Ce cahier complète le formulaire **Mes talents**.

Un talent n'est pas forcément spectaculaire.

C'est souvent quelque chose que tu fais naturellement bien, avec une certaine fluidité, parfois sans mesurer sa valeur.

Ce cahier t'aide à repérer ce qui revient souvent, ce qui te donne de l'énergie et ce que les autres remarquent chez toi.

## Pour l'accompagnant

Cette section n'est pas destinée à la participante.

À l'issue de ce cahier, chercher à identifier :

- les facilités naturelles ;
- les talents reconnus par autrui ;
- les talents énergisants ;
- les talents sous-estimés ;
- les talents sous-utilisés ;
- les liens avec compétences, motivations et besoins.

Ne pas présenter les talents comme des certitudes.

Le bon vocabulaire est : « il semble que », « plusieurs indices indiquent », « c'est une hypothèse à confirmer ».
