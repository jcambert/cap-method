# Cahier de réflexion – Module 9 : Mes ressources et contraintes

**Version :** 1.0

---

## Présentation

Ce cahier complète le formulaire **Mes ressources et contraintes**.

Il sert à regarder la réalité sans se décourager.

Un projet solide ne se construit pas uniquement sur des envies.

Il se construit aussi avec les ressources disponibles, les contraintes à respecter et les éléments à sécuriser.

L'objectif n'est pas de réduire tes possibilités.

L'objectif est de construire des scénarios réalistes et soutenables.

## Pour l'accompagnant

Cette section n'est pas destinée à la participante.

À l'issue de ce cahier, chercher à identifier :

- les ressources sous-estimées ;
- les soutiens mobilisables ;
- les contraintes fortes ;
- les contraintes négociables ;
- les contraintes non modifiables à court terme ;
- les éléments à sécuriser ;
- les premiers pas réalistes.

Ne pas utiliser les contraintes pour fermer les possibilités trop tôt.

Les contraintes servent à construire des scénarios progressifs et sécurisés.
