# LIVRABLE-087
# Index des Google Forms CAP

**Version :** 1.0

---

## Objectif

Cet index centralise la liste des formulaires CAP à créer et à utiliser pendant un accompagnement.

Il permet de suivre rapidement :

- le nom du formulaire ;
- le module concerné ;
- le livrable source ;
- le temps estimé ;
- le statut de création ;
- le lien Google Forms ;
- le lien Google Sheets associé.

## Modèle de nommage des fichiers Forms

Nom recommandé :

```text
CAP – Module XX – Nom du module – Prénom participante
```

Exemple :

```text
CAP – Module 04 – Mes besoins – Claire
```

Pour un modèle réutilisable :

```text
CAP – TEMPLATE – Module 04 – Mes besoins
```

## Points de vigilance

Ne pas envoyer tous les formulaires en une fois.

Ne pas créer un parcours trop lourd pour la participante.

Ne pas utiliser les formulaires comme des tests de notation.

Ne pas conclure uniquement à partir des scores.

---

## Résultat attendu

Cet index doit devenir la table de pilotage des formulaires CAP.

Il permet à l'accompagnant de savoir à tout moment :

- quel formulaire existe ;
- lequel est prêt ;
- lequel a été envoyé ;
- où sont les réponses ;
- quelle étape vient ensuite.
