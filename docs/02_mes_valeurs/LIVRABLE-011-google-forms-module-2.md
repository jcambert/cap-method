# LIVRABLE-011
# Google Forms – Module 2 : Mes valeurs

**Version :** 1.0

---

## Informations générales

**Nom du formulaire**

CAP – Module 2 : Mes valeurs

**Objectif**

Identifier les valeurs importantes de la participante, comprendre leur origine, leur niveau de satisfaction actuel et les éventuels écarts entre ce qui compte pour elle et ce qu'elle vit aujourd'hui.

**Temps estimé**

35 à 50 minutes

---

## Paramètres Google Forms

- Barre de progression : Activée
- Questions obligatoires : Oui, sauf mention contraire
- Retour arrière autorisé : Oui
- Modification des réponses : Oui
- Sections conditionnelles : Non obligatoire pour la version 1.0

---

## SECTION 1 – Introduction

### Texte d'introduction

Les valeurs représentent ce qui est profondément important pour toi.

Elles influencent tes choix, tes relations, ton énergie et parfois tes frustrations.

Certaines valeurs sont très visibles. D'autres apparaissent surtout lorsqu'elles ne sont pas respectées.

Dans ce module, nous allons chercher à mieux comprendre ce qui compte vraiment pour toi aujourd'hui.

Il n'y a pas de bonne ou de mauvaise réponse.

## Résultat attendu

À la fin de ce module, l'accompagnant doit pouvoir identifier :

- les valeurs principales ;
- les valeurs non négociables ;
- les valeurs actuellement bien respectées ;
- les valeurs en tension ;
- les premiers écarts entre ce qui compte pour la participante et son quotidien actuel.
