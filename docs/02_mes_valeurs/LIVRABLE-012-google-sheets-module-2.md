# LIVRABLE-012
# Google Sheets – Module 2 : Mes valeurs

**Version :** 1.0

---

## Objectif

Ce classeur reçoit les réponses du Google Forms **Module 2 – Mes valeurs**.

Il permet de structurer les valeurs principales, d'identifier les écarts entre importance et satisfaction, de préparer les hypothèses et de générer les éléments utiles à la restitution intermédiaire.

Le classeur ne décide jamais à la place de l'accompagnant.

Il prépare une lecture structurée.

---

## Architecture du classeur

Le classeur est composé de neuf feuilles.

1. Réponses
2. Référentiel valeurs
3. Sélection
4. Priorisation
5. Alignement
6. Hypothèses
7. Restitution
8. Tableau de bord
9. Journal accompagnant

## Feuille 2 — Référentiel valeurs

Cette feuille contient toutes les valeurs disponibles dans CAP.

Colonnes :

- Valeur
- Famille
- Description courte
- Exemples de comportements associés
- Risque si non respectée
- Questions liées

### Familles proposées

- Relations humaines
- Développement personnel
- Travail
- Société
- Réalisation
- Style de vie

### Exemples

| Valeur | Famille | Description courte | Risque si non respectée |
|---|---|---|---|
| Autonomie | Style de vie | Pouvoir décider et organiser son action | Sentiment d'étouffement |
| Respect | Relations humaines | Être considéré avec dignité | Colère, retrait, perte de confiance |
| Transmission | Relations humaines | Partager, expliquer, faire progresser | Perte de sens |
| Sécurité | Style de vie | Stabilité, prévisibilité, cadre | Anxiété, hésitation |

---

## Feuille 4 — Priorisation

Objectif : ordonner les valeurs selon leur importance.

Colonnes :

- Participant
- Rang
- Valeur
- Famille
- Importance estimée
- Indice de stabilité
- Preuves associées

### Indice de stabilité

À renseigner manuellement :

- Faible : valeur peu illustrée
- Moyen : valeur citée et illustrée une fois
- Fort : valeur citée, priorisée et illustrée par plusieurs exemples

---

## Feuille 5 — Alignement

Objectif : mesurer l'écart entre importance et satisfaction actuelle.

Colonnes :

- Participant
- Valeur
- Importance
- Satisfaction actuelle
- Écart
- Niveau de tension
- Domaine concerné
- Action possible

### Formule proposée

`Écart = Importance - Satisfaction actuelle`

### Niveau de tension

- 0 à 2 : Tension faible
- 3 à 5 : Tension modérée
- 6 à 10 : Tension forte

## Mise en forme

- Vert : valeurs satisfaites
- Orange : tensions modérées
- Rouge : tensions fortes
- Bleu : commentaires de l'accompagnant
- Violet : hypothèses à confirmer dans d'autres modules

---

## Contrôle qualité

Avant de clôturer le module, vérifier :

- [ ] Les valeurs sélectionnées sont correctement reprises.
- [ ] Les Top 10 et Top 5 sont identifiés.
- [ ] La valeur principale est documentée.
- [ ] Les valeurs en tension sont repérées.
- [ ] Les hypothèses sont prudentes et argumentées.
- [ ] La restitution est préparée.

## Livrables générés

À partir de ce classeur, l'accompagnant dispose de :

- les valeurs en tension ;
- les premières hypothèses liées aux conflits de valeurs ;
- les éléments à intégrer dans la restitution intermédiaire.
