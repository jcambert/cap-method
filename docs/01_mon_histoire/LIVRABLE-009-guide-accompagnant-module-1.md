# LIVRABLE-009
# Guide de l'accompagnant – Module 1 : Mon histoire

**Version :** 1.0

---

## Objectif du guide

Ce guide accompagne l'analyse du premier module.

Son objectif n'est pas de trouver immédiatement des réponses.

Son objectif est de construire les premières hypothèses qui seront confirmées, nuancées ou abandonnées au fil des modules suivants.

Le Module 1 est un module d'observation.

Il ne sert pas à orienter la participante.

---

## Avant de commencer

Prévois environ 45 à 60 minutes.

Installe-toi dans un environnement calme.

Lis une première fois les réponses sans prendre de notes.

Cherche simplement à comprendre l'histoire.

Seulement lors de la seconde lecture, commence à relever des éléments importants.

---

## Les cinq questions à garder en tête

Pendant la lecture, demande-toi systématiquement :

1. Quels événements semblent avoir le plus influencé son parcours ?
2. Quelles ressources ressortent déjà ?
3. Quels apprentissages la participante met-elle en avant ?
4. Quels thèmes reviennent plusieurs fois ?
5. Quels sujets mériteront d'être approfondis plus tard ?

---

## Ce qu'il faut rechercher

### Les événements fondateurs

Repère les moments qui semblent avoir changé durablement sa manière de voir les choses.

Exemples :

- une rencontre ;
- un changement de métier ;
- une réussite ;
- un échec ;
- une maladie ;
- un déménagement ;
- une naissance ;
- un engagement associatif.

### Les ressources

À chaque fois que la participante raconte une difficulté, demande-toi :

**Qu'a-t-elle mobilisé pour avancer ?**

Les ressources peuvent être :

- une compétence ;
- une qualité ;
- un soutien extérieur ;
- une capacité d'adaptation ;
- de la persévérance.

### Les répétitions

Repère les mots ou thèmes qui reviennent.

Exemples :

- aider ;
- apprendre ;
- transmettre ;
- organiser ;
- réparer ;
- créer.

Ne conclus pas.

Note simplement les répétitions.

---

## Les questions de relance

Si un échange est prévu après le formulaire, privilégie des questions ouvertes.

Exemples :

- Quel événement t'a le plus transformée ?
- Pourquoi celui-ci plutôt qu'un autre ?
- Qu'est-ce que cette expérience t'a appris sur toi ?
- Quelles qualités t'ont permis d'avancer ?

Évite les questions qui suggèrent une réponse.

---

## Les points de vigilance

Ne confonds jamais :

- un fait avec une interprétation ;
- une émotion avec une valeur ;
- une réussite avec un talent.

Le Module 1 ne permet pas de conclure.

Il permet uniquement d'observer.

---

## Les premiers fils rouges

Commence une liste.

Un fil rouge est un thème qui apparaît plusieurs fois.

| Thème | Nombre d'occurrences | Niveau de confiance |
|---|---:|---|
| Transmission | 2 | À explorer |
| Créativité | 1 | Faible |
| Organisation | 3 | Modéré |

Cette liste évoluera tout au long du parcours.

---

## Les hypothèses

Rédige les hypothèses avec prudence.

### Bon exemple

Plusieurs éléments laissent penser que la participante apprécie les situations où elle peut aider les autres. Cette hypothèse sera vérifiée dans les modules Valeurs, Motivations et Talents.

### À éviter

Cette personne est faite pour l'accompagnement.

---

## Les erreurs fréquentes

Ne pas :

- interpréter une seule réponse ;
- chercher un métier dès le premier module ;
- vouloir tout expliquer ;
- remplir tous les champs à tout prix.

Il est normal de terminer ce module avec des zones d'incertitude.

---

## Préparer la restitution

La restitution doit toujours comporter quatre parties.

### 1. Ce qui ressort déjà

Présenter les premiers éléments observés.

### 2. Les ressources identifiées

Mettre en avant les forces visibles dans le parcours.

### 3. Les questions encore ouvertes

Expliquer ce qui sera exploré plus tard.

### 4. Le lien avec le prochain module

Créer une transition naturelle vers **Mes valeurs**.

---

## Critères de validation du module

Le module est validé lorsque :

- [ ] Les événements majeurs sont identifiés.
- [ ] Les ressources sont listées.
- [ ] Les premiers fils rouges sont notés.
- [ ] Les hypothèses sont rédigées.
- [ ] Les questions à approfondir sont préparées.
- [ ] La restitution est prête.

## Fin du Module 1

Lorsque ce guide est complété, le Module 1 est considéré comme terminé.
