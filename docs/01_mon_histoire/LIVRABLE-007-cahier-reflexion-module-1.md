# Cahier de réflexion – Module 1 : Mon histoire

**Version :** 1.0

---

## Présentation

Ce cahier complète le questionnaire.

Le formulaire permet de recueillir des informations.

Le cahier t'invite à prendre davantage de recul.

Prends ton temps.

Tu peux compléter ces pages en plusieurs fois.

Tu peux écrire quelques lignes ou plusieurs pages.

Il n'existe aucune bonne ou mauvaise réponse.

## Pour l'accompagnant

Cette section n'est pas destinée à la participante.

### Objectifs pédagogiques

À travers ce cahier, chercher à faire émerger :

- les premiers fils rouges du parcours ;
- les ressources personnelles ;
- les événements fondateurs ;
- les apprentissages majeurs ;
- les croyances positives ou limitantes qui pourront être explorées plus tard.

Ce cahier ne doit jamais être évalué.

Il constitue un support de réflexion et de dialogue.
