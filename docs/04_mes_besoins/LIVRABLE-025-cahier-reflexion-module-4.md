# Cahier de réflexion – Module 4 : Mes besoins

**Version :** 1.0

---

## Présentation

Ce cahier complète le formulaire **Mes besoins**.

Les besoins sont parfois moins faciles à identifier que les valeurs.

On les remarque souvent lorsqu'ils ne sont pas respectés : fatigue, frustration, retrait, tension, perte d'envie.

Ce cahier t'aide à observer ce qui te nourrit, ce qui t'épuise et ce dont tu as besoin pour avancer durablement.

## Pour l'accompagnant

Cette section n'est pas destinée à la participante.

À l'issue de ce cahier, chercher à identifier :

- les besoins fortement exprimés ;
- les besoins déjà nourris ;
- les besoins en tension ;
- les besoins non assumés ;
- les liens avec les valeurs et le fonctionnement.

Ne pas transformer un besoin en exigence ou en injonction.

La restitution doit ouvrir une compréhension, pas prescrire une solution.
