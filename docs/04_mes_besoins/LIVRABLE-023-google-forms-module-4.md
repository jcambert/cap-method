# LIVRABLE-023
# Google Forms – Module 4 : Mes besoins

**Version :** 1.0

---

## Informations générales

**Nom du formulaire**

CAP – Module 4 : Mes besoins

**Objectif**

Identifier les besoins essentiels de la participante : ce qui soutient son équilibre, son énergie, sa motivation et son sentiment d'être à sa place.

Ce module distingue les envies, les préférences et les besoins profonds.

**Temps estimé**

40 à 55 minutes

---

## Paramètres Google Forms

- Barre de progression : Activée
- Questions obligatoires : Oui, sauf mention contraire
- Retour arrière autorisé : Oui
- Modification des réponses : Oui
- Échelles harmonisées : 1 à 10 lorsque nécessaire

---

## SECTION 1 – Introduction

### Texte d'introduction

Dans ce module, nous allons explorer tes besoins.

Un besoin n'est pas forcément une envie.

Une envie peut être ponctuelle.

Un besoin est plus profond : lorsqu'il n'est pas respecté longtemps, ton énergie, ton équilibre ou ta motivation peuvent diminuer.

L'objectif n'est pas de te juger.

L'objectif est de mieux comprendre ce dont tu as besoin pour avancer durablement.

## Résultat attendu

À la fin de ce module, l'accompagnant doit pouvoir identifier :

- les besoins essentiels ;
- les besoins actuellement satisfaits ;
- les besoins en tension ;
- les besoins insuffisamment assumés ;
- les liens entre besoins, valeurs et fonctionnement.

Aucune prescription de changement ne doit être formulée sans validation par la participante.
