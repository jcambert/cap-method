# LIVRABLE-073
# Grille de validation – Synthèse finale CAP

**Version :** 1.0

---

## Objectif

Cette grille permet à l'accompagnant de vérifier que la synthèse finale CAP est complète, prudente, exploitable et fidèle au parcours réalisé.

Elle doit être utilisée avant d'envoyer la synthèse à la participante.

---

|---|---|---|---|

| Les fils rouges sont identifiés | | | |
| Les éléments importants ne sont pas noyés dans les détails | | | |
| La synthèse est lisible pour la participante | | | |

## 7. Formulations interdites ou à éviter

Vérifier qu'aucune formule de ce type n'apparaît :

- Tu es faite pour...
- Ton vrai profil est...
- Ta vocation est...
- Tu dois absolument...
- Tu n'es pas capable de...
- Ce choix est le bon.
- Ce scénario est impossible.

---

## 8. Formulations recommandées

Préférer :

- Plusieurs éléments semblent indiquer que...
- Cette piste paraît cohérente à ce stade.
- Ce point mérite d'être confirmé.
- Ce scénario pourrait être testé progressivement.
- Cette contrainte devra être sécurisée.
- Cette ressource peut soutenir les premiers pas.

---

## 9. Validation finale

La synthèse peut être envoyée si :

- [ ] elle est complète ;
- [ ] elle est prudente ;
- [ ] elle est concrète ;
- [ ] elle valorise la participante ;
- [ ] elle ne décide pas à sa place ;
- [ ] elle débouche sur une action réaliste.
