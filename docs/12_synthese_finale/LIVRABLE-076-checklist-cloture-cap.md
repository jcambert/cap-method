# LIVRABLE-076
# Checklist de clôture CAP

**Version :** 1.0

---

## Objectif

Cette checklist permet de clôturer proprement un parcours CAP.

Elle vérifie que la participante repart avec :

- une synthèse claire ;
- un scénario prioritaire ;
- un plan d'action ;
- une première action ;
- une date de révision ;
- des points de vigilance ;
- une autonomie suffisante pour continuer.

- [ ] la synthèse finale ;
- [ ] le plan d'action ;
- [ ] la première action ;
- [ ] la date de révision ;
- [ ] la possibilité d'ajuster le plan ;
- [ ] une vision claire des prochaines étapes.

---

## 7. Posture de clôture

La clôture doit rappeler que :

- le parcours a produit une base de réflexion ;
- la participante reste libre de ses choix ;
- le scénario prioritaire n'est pas une obligation ;
- le plan est révisable ;
- les petits pas comptent ;
- la suite se construira avec l'expérience.

---

## 8. Message final type

Bonjour **[Prénom]**,

Nous arrivons à la clôture du parcours CAP.

Tu repars avec une synthèse de ce que tu as découvert, un scénario prioritaire provisoire et un plan d'action concret.

Ce plan n'est pas figé.

Il est fait pour être testé, ajusté et enrichi avec ce que tu apprendras en avançant.

Le plus important maintenant est de poser le premier pas prévu, puis d'observer ce qu'il t'apprend.

Bravo pour le travail réalisé.
