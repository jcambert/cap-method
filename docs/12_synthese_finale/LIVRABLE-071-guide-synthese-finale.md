# LIVRABLE-071
# Guide – Synthèse finale CAP

**Version :** 1.0

---

## Objectif

Ce guide permet de préparer la synthèse finale du parcours CAP.

La synthèse finale rassemble les éléments essentiels des modules précédents afin de produire une vision claire, utile et actionnable pour la participante.

Elle ne doit pas être présentée comme un diagnostic, un test psychologique ou une décision définitive.

Elle doit rester une restitution structurée, prudente et orientée vers l'action.

---

## Rôle de la synthèse finale

La synthèse finale sert à :

- rappeler le chemin parcouru ;
- identifier les fils rouges ;
- clarifier les valeurs, besoins, motivations, compétences et talents ;
- reformuler l'environnement favorable ;
- présenter le scénario prioritaire ;
- rappeler le plan d'action ;
- préciser les points de vigilance ;
- encourager la participante à continuer après l'accompagnement.

## Structure recommandée

La synthèse finale doit contenir les sections suivantes :

1. Introduction
2. Chemin parcouru
3. Fils rouges du parcours
4. Valeurs principales
5. Besoins essentiels
6. Fonctionnement favorable
7. Motivations profondes
8. Compétences et talents
9. Environnement idéal
10. Ressources et contraintes
11. Scénario prioritaire
12. Plan d'action
13. Points de vigilance
14. Prochaines étapes
15. Message de clôture

## Principes de rédaction

### Formuler avec prudence

Utiliser :

- il semble que ;
- plusieurs éléments indiquent ;
- cette hypothèse ressort ;
- ce point mérite d'être confirmé ;
- cette piste paraît cohérente à ce stade.

Éviter :

- tu es ;
- ton profil est ;
- tu dois ;
- ta vocation est ;
- le bon choix est.

---

## Fils rouges à identifier

Un fil rouge est un élément qui revient dans plusieurs modules.

Exemples :

- besoin d'autonomie présent dans les valeurs, le fonctionnement et l'environnement idéal ;
- motivation d'aider ou transmettre confirmée par les compétences et talents ;
- besoin de sécurité lié aux contraintes et au plan d'action ;
- fatigue liée aux environnements flous ou relationnellement tendus.

---

## Méthode de synthèse

### Étape 1 – Extraire les éléments forts

Pour chaque module, noter trois éléments maximum.

### Étape 2 – Repérer les répétitions

Repérer les éléments présents dans plusieurs modules.

### Étape 3 – Identifier les tensions

Exemples :

- envie de changement et besoin de sécurité ;
- autonomie forte et besoin de cadre ;
- compétence maîtrisée mais peu motivante ;
- scénario désirable mais peu sécurisé.

### Étape 4 – Formuler les hypothèses finales

Les hypothèses finales doivent être utiles pour agir.

Elles ne doivent pas enfermer la participante.

### Étape 5 – Relier au plan d'action

La synthèse finale doit déboucher sur les prochaines étapes concrètes.

---

## Questions de contrôle

Avant d'envoyer la synthèse finale, vérifier :

- La participante se reconnaîtra-t-elle probablement dans cette synthèse ?
- Les formulations restent-elles prudentes ?
- Les forces sont-elles suffisamment valorisées ?
- Les contraintes sont-elles traitées sans fatalisme ?
- Le scénario prioritaire reste-t-il révisable ?
- Le plan d'action est-il concret ?
- La synthèse donne-t-elle envie d'avancer ?

---

## Points de vigilance

Ne pas produire une synthèse trop longue.

Ne pas recopier toutes les réponses.

Ne pas transformer la synthèse en analyse psychologique.

Ne pas proposer un métier comme conclusion unique.

Ne pas effacer les nuances.

Ne pas oublier le plan d'action.

---

## Critères de validation

La synthèse finale est validée lorsque :

- [ ] Les modules 1 à 11 sont pris en compte.
- [ ] Les fils rouges sont identifiés.
- [ ] Les hypothèses sont prudentes.
- [ ] Le scénario prioritaire est présenté.
- [ ] Le plan d'action est rappelé.
- [ ] Les prochaines étapes sont concrètes.
- [ ] Le document peut être remis à la participante.
