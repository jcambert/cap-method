# Cahier de réflexion – Module 11 : Mon plan d'action

**Version :** 1.0

## Présentation

Ce cahier complète le formulaire **Mon plan d'action**.

Il sert à transformer une piste en mouvement concret.

Un plan d'action n'est pas une promesse rigide.

C'est une feuille de route simple, progressive et révisable.

L'objectif est d'avancer sans se mettre en danger, en testant, en sécurisant et en ajustant.

## Pour l'accompagnant

Cette section n'est pas destinée à la participante.

À l'issue de ce cahier, vérifier que :

- le plan n'est pas trop lourd ;
- la première action est simple ;
- les risques sont pris en compte ;
- le test terrain est concret ;
- la date de révision est fixée ;
- la participante reste actrice de ses choix.

Ne pas chercher à accélérer artificiellement.

Un bon plan d'action doit être tenable.
