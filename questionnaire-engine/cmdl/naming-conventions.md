# CMDL Naming Conventions

## Form ids

Use `FORM-001` format.

## File names

Use `<FORM-ID>.cmdl.yaml`.

Examples:

- FORM-001.cmdl.yaml
- FORM-002.cmdl.yaml

## Section ids

Use uppercase semantic names.

Examples:

- WELCOME
- PERSONAL_INFORMATION
- PROFESSIONAL_SITUATION

## Question ids

Use stable question ids inside each form.

Examples:

- Q001
- Q002
- Q003

## Stability rule

Never reuse a question id for a different meaning.

## Language

Use `fr-FR` for the first CAP Method release.
