# Analysis Snapshot

## Purpose

`AnalysisSnapshot` is the first structured analysis output produced from a normalized `ResponseSession`.

It is designed for consultant review, not for automatic final decision making.

## Input

```text
ResponseSession JSON
```

Generated by:

```bash
node questionnaire-engine/tools/import-response-session.mjs
```

## Output

```text
AnalysisSnapshot JSON
```

Generated by:

```bash
node questionnaire-engine/tools/analyze-response-session.mjs
```

## Default command

```bash
node questionnaire-engine/tools/analyze-response-session.mjs
```

## Custom command

```bash
node questionnaire-engine/tools/analyze-response-session.mjs \
  questionnaire-engine/responses/generated/session.response.normalized.json \
  questionnaire-engine/analysis/generated/sample.analysis-snapshot.json
```

## Structure

```yaml
id: string
sessionId: string
beneficiaryId: string
consultantId: string
generatedAt: datetime
source: response_session
status: ready_for_consultant_review | incomplete
coverage:
  formsImported: number
  formsExpected: number
  answersImported: number
  complete: boolean
  analysisReady: boolean
indicators: AnalysisIndicator[]
highlights: AnalysisHighlight[]
risks: AnalysisRisk[]
consultantPrompts: ConsultantPrompt[]
traceability:
  sourceSessionId: string
  formIds: string[]
```

## Current indicators

- form completion rate;
- rating5 average;
- rating10 average;
- long text answer count;
- analysis readiness.

## Current highlights

The first version detects simple qualitative signals around:

- skills;
- career evolution;
- constraints;
- confidence and projection.

## Current risks

The first version detects:

- low average ratings;
- limited qualitative material.

## Consultant prompts

The snapshot produces review questions such as:

- what expectations must be clarified;
- which strengths seem confirmed;
- which constraints may limit professional options;
- which risks should be addressed first.

## Important rule

The analysis snapshot is a support for consultant interpretation.

It is not a final bilan de competences synthesis.

Final synthesis requires:

- consultant validation;
- narrative consolidation;
- explicit link with objectives;
- action plan formulation.
